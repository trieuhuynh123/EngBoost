# 📚 TOEIC-IELTS Learning Test Online 🎓

Welcome to **TOEIC-IELTS Learning Test Online**! 🚀 A comprehensive web platform designed to help you prepare for your TOEIC and IELTS exams. With an interactive learning environment, practice tests, and progress tracking, this platform is your perfect companion in your exam preparation journey. 📖

---

## Features 🌟

- **Interactive Practice Tests:** Take TOEIC and IELTS practice tests and get immediate feedback. 📝
- **Progress Tracking:** Monitor your improvement over time with detailed analytics and insights. 📊
- **Timed Tests:** Simulate real exam conditions with timed practice tests. ⏱️
- **Question Bank:** Access a wide range of questions covering listening, reading, writing, and speaking sections. 🎧📚✍️
- **User Accounts & Profiles:** Create a personal profile, track your test history, and manage your learning progress. 💼
- **Responsive Design:** Fully responsive interface for desktop, tablet, and mobile devices. 📱💻
- **Admin Panel:** For managing questions, tests, and user progress. 🔧

## Tech Stack 🚀

- **Frontend:** Next.js, React, Styled Components, Tailwind CSS
- **Backend:** NestJS, TypeORM, PostgreSQL
- **Authentication:** JWT, OAuth2
- **Hosting:** Vercel, Heroku (for the frontend), DigitalOcean (for the backend)
- **APIs:** REST API for integration with the frontend
